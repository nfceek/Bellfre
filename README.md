# Bellfre

**Bellfre** is an experimental open-source project exploring a simple question:

**How do AI agents find and communicate with each other?**

As AI moves beyond chatbots into autonomous software and business systems, there is an emerging need for a lightweight way for agents to advertise their capabilities, publish metadata, and become discoverable by other applications and AI agents. Bellfre is an early exploration of that discovery layer.

The project proposes a simple, developer-friendly **Agent Manifest** that describes an agent's identity, capabilities, supported protocols, authentication methods, and connection endpoints. Rather than acting as another AI model, Bellfre aims to become a discovery service—a beacon that helps people and software locate the right agent for the right task.

Long term, Bellfre may evolve into an open registry supporting searchable agent catalogs, business AI services, capability-based discovery, and interoperability with emerging standards such as MCP, A2A, and future agent communication protocols.

The project is currently in the research and prototype stage. The initial goal is to define a practical, extensible specification, build a reference implementation in Python, and encourage community discussion around open AI agent discovery.

Contributions, feedback, and ideas are welcome as the project evolves.
